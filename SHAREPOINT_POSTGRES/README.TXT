

Mediante este script en Python  he trabajado sobre un dataset csv ('https://www.kaggle.com/datasets/uciml/electric-power-consumption-data-set') de consumos eléctricos diarios de aproximadamente 2.000.000 de registros para integrarlos después de hacer alguna transformación en una base PostgreSQL. También he manipulado esos registros creando varios archivos para moverlos hacia y desde un sitio Sharepoint. EL archivo orgiginal de registros lo he limitado a los datos de 2009 (525.000) dados los límites de Github. En ese mismo script también desarrollo funcionalidades en relación a manejo de archivos en Sharepoint (desde y hacia) por si los datos iniciales  se encuentran en ese entorno Microsoft.
